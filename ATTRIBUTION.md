## üìã Project Information

**Project Name:** YOLOv12-3D: 3D Object Detection for KITTI Dataset

**Description:** An extension of YOLOv12 for 3D object detection, specifically designed for autonomous driving applications using the KITTI benchmark dataset.

**Key Features:**
- Native KITTI dataset support
- End-to-end 3D bounding box prediction
- Based on state-of-the-art YOLOv12 architecture
- Multiple model sizes for different use cases
- Complete training and inference pipeline
